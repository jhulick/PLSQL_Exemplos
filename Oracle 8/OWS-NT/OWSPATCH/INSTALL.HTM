Oracle WebServer

Enhanced Security Utilities

Windows NT Installation

  • Download the file owspatch.exe using binary transfer mode. Save the file in a temporary directory (for instance c:\temp)
  • Launch the self-extracting archive by either double-clicking on the file in File Manager, or in a DOS shell, type: 
cd c:\temp\patch
owspatch.exe

This will extract all the files into the directory. Please refer to the PATCH.doc file for further instructions on enhancing security for Oracle WebServer 1.0 or 2.0 on Windows NT.

Files included in the Oracle WebServer for Windows NT self-extracting archive

owspatch.exe

  • patchv1.exe - The security patch for Oracle Webserver 1.0
  • patchv1.exe - The security patch for Oracle Webserver 2.0.x
  • diagn1.exe - The diagnostic utility for Oracle Webserver 1.0
  • diagn2.exe - The diagnostic utility for Oracle Webserver 2.0.x
  • PATCH.DOC - The document containing detailed instructions on security enhancement.
  • MSVCRT.DLL - Support C runtime DLL.
