LiveHTML Cartridge Configuration

Enable LiveHTML: Setting this switch enables parsing of HTML documents with LiveHTML tags and processing them. Disabling it would return the document unprocessed 

        Enable LiveHTML Processing
        Disable LiveHTML processing

Enable .html Extensions: By default only files with .shtml extension are processed. (provided the EnableLiveHTML switch is ON) By enabling this switch files with non-shtml extensions are also processed. 

        Enable .html extensions
        Disable .html extensions
Extension List: This list specifies the extensions to the files that need to be parsed. This list is consulted only if the previous switch--Mo was disabled.

Enable exec Tag: The exec tag allows executing commands from within the HTML document and can potentially be a security loophole. Disabling this will not allow processing of the exec tag. 

  
        Enable processing exec tag
        Disable processing exec tag
