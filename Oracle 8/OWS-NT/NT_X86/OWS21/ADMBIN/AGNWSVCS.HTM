Create New DCD (Database Connection Descriptor

From this screen, you can create a new Database Connection Descriptor. Create a new DCD by filling in the fields below, selecting to create a new PL/SQL Agent Database User, install the WebServer Developer's Toolkit, and then selecting the "Submit New Service" button.

For detailed information on each of the fields below, follow this Help link, or select the Help button below.

Note that, for security, after submitting the new service, the password fields will be cleared.
