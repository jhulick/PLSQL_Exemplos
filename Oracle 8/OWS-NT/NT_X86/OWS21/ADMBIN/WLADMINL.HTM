Oracle Web Listener Home Page

From here, you can do the following:
  • Create a new Oracle Web Listener to run on your machine by clicking the Create Listener... button.

  • Modify existing Oracle Web Listeners by selecting the Configure link below.

  • Start, Stop or Delete existing Oracle Web Listeners by selecting the appropriate link below.

  • Learn how to get a digital certificate for your WebServer so that you can use SSL to encrypt datastreams for a safer connection. Oracle WebServer initially supports certificates issued by Verisign.
The Oracle Web Listener can listen on multiple ports. You can create a Listener configured to run on a single port or multiple ports. This choice is merely to make the listener creation process very simple for the single-port case. After you have created a listener, the distinction no longer exists.
