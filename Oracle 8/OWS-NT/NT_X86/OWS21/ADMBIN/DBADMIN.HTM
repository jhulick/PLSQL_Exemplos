Startup or Shutdown a Database

From this screen, you can startup or shutdown any of your Oracle7 databases, provided that the database is running on the same machine as your Oracle Web Listener. Just select the database and then the appropriate action. If there are no databases listed, or to find out what the different startup/shutdown modes do, then select this Help link, or the Help button below for more information.
