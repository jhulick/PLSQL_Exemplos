Modify DCD (Database Connection Descriptor)

To copy an existing Database Connection Descriptor (DCD), change the DCD name and then select the "Copy Service" button. Note that when you copy an existing DCD, if you do not specify a new password, the password will be copied.

For any other changes to the DCD, select the "Modify Service" button.

To delete this DCD, select the "Delete Service" button.
