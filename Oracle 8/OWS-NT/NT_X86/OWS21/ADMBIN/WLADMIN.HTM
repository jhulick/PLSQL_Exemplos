Oracle Web Listener Administration

You should currently be using the Oracle Web Listener Administration Server to access this page. You should now configure your own Oracle Web Listener.

The Oracle Web Listener can listen on multiple ports. You can create a Listener configured to run on a single port or multiple ports. This choice is merely to make the listener creation process very simple for the single-port case. After you have created a listener, the distinction no longer exists.
