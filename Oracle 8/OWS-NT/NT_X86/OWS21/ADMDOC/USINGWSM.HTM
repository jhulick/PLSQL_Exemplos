Using the Oracle WebServer Manager

The Oracle WebServer Manager is a collection of HTML forms you can use to configure your Oracle WebServer. These forms are divided into the following sections:

Oracle7 Database Administration

Using the Oracle7 Database Administration page, you can:

  • Start up an Oracle7 database
  • Shut down an Oracle7 database

Oracle Web Listener Administration

Using the Web Listener Administration pages, you can:

  • Create a new Web Listener
  • Configure Web Listener network parameters
  • Configure Web Listener logging parameters
  • Configure Web Listener user and group parameters
  • Configure Web Listener virtual directory mappings
  • Configure Web Listener file caching
  • Define Web Listener file extensions for language formats
  • Define Web Listener file extensions for MIME types
  • Define Web Listener file extensions for encoding formats
  • Configure Web Listener security
  • Control access to virtual files and directories
  • Configure port security (SSL) (to learn how to obtain a certificate, see Setting Up a Secure Oracle WebServer)
  • Start a Web Listener process
  • Stop a Web Listener process
  • Delete a Web Listener
  • Enable Web Listener proxy behavior
  • Enable LiveHTML parsing for a Web Listener

External Web Listener Registration

Using the External Web Listener Registration page, you can:

  • Register external Web Listeners
  • Configure Oracle WebServer 2.1 to work with the Netscape FastTrack Server

PL/SQL Agent Administration

Using the PL/SQL Administration pages, you can:

  • Create a new Database Connection Descriptor (DCD)
  • Create a default DCD
  • Create a DCD for database administration
  • Create a new DCD from an existing configuration
  • Modify a DCD
  • Delete a DCD

Web Request Broker Administration

Using the Web Request Broker Administration pages, you can:

  • Configure a new WRB cartridge
  • Modify a WRB cartridge configuration
  • Delete a WRB cartridge
