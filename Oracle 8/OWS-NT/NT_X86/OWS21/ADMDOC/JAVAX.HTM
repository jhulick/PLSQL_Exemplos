Server-Side Java Examples

This section provides examples of how to write applications for the Java cartridge using the Java Web Toolkit. It is divided into the following areas:

  • Example of Package Wrapper Generation. A Java application that is to access the database using PL/SQL must have class wrappers for all the PL/SQL packages that it accesses. This shows how to run the pl2java utility to create the appropriate wrappers.
  • Example of Java Database Access. This shows how a Java application accesses the Oracle database and interfaces to PL/SQL.
  • Java Dynamic HTML Examples. This shows the effect of various Java Web Toolkit classes designed to generate HTML dynamically.
