You have reached the Oracle WebServer Database Browser Error Page most likely because your Oracle WebServer installation was incomplete.

In order to install the Oracle WebServer Database Browser, go to the Oracle WebServer Installation page.

If you have already installed the Oracle WebServer Database Browser, and you have reached this page, then the check the owa_dba.err error file which can be found in the default Oracle WebServer log directory.

Once you have checked the error file, you may find that the OWA_DBA service is not correctly configured. If this is the case, then you can either go back to the Oracle WebServer Installation page to re-install the Oracle WebServer Database Browser. Or, you may want to go the the Oracle Web Agent Configuration page, and then modify the OWA_DBA service.
