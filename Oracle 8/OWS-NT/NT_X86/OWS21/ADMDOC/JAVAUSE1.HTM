Java Web Toolkit Overview

Oracle WebServer provides a set of Java classes to enable you to access the database and to generate HTML dynamically using the Java Cartridge. These classes are called the Java Web Toolkit. Using these Java classes, you can make PL/SQL calls from within your Java application, effectively combining the strengths of the two languages. This toolkit also includes a type wrapper for Java applets, to make it easy for you to store applets in the database, retrieve them through the Java Cartridge, and send them to the client for execution embedded in a Web page.

To use the Java Web Toolkit, import the following into your Java code:

  • oracle.html.* contains the objects for dynamic HTML generation.
  • oracle.rdbms.* contains the objects for database access.
  • oracle.plsql.* contains the objects for PL/SQL access.
Obviously, if you only want to perform some of these functions, you need only import some of these objects.
