Example of PL/SQL Agent Parameter Passing Using POST

The following example uses an HTML form that employs the POST request method.

<FORM METHOD="POST" ACTION="http://www.blare.com:8080/member/owa/music_lookup">

Please type the name of the artist you wish to search for:

<INPUT TYPE="text" NAME="artist"><P> To submit the query, press this button: <INPUT TYPE="submit" VALUE="Submit Query">. <P> </FORM>

The above form does the same thing as the Example of PL/SQL Agent Parameter Passing Using GET, except that, instead of populating the QUERY_STRING environment variable with "artist=Ellington," the Web Listener writes "artist=Ellington" to standard input.
