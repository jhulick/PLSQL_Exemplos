Example PL/SQL Agent Web Pages

This section provides examples of how to create simple Web Pages using the packages in the PL/SQL Web Toolkit. The QUERY_FORM example is the most complete. The others focus on the areas where they differ from QUERY_FORM.

The examples are listed below:

  • The General PL/SQL Web Page Example shows how to use several of the PL/SQL Web Toolkit procedures to create a dynamic Web page.
  • The QUERY_FORM PL/SQL Web Page Example prints an HTML page using all the columns of a specified table. It shows you how to deal with PL/SQL tables representing multivalued attributes.
  • The DO_QUERY PL/SQL Web Page Example is a variation on QUERY_FORM. It let's the user select the desired columns.
  • The OVERLOAD PL/SQL Web Page Example shows you how to create a package containing overloaded functions and procedures.
  • The Multivalued Parameter PL/SQL Web Page Example shows you how to pass a variable number of selected items from a form field to a procedure as a PL/SQL table.
  • The Cookie PL/SQL Example shows you how to use HTTP cookies in your application.
