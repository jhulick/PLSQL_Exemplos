Configuring a New WRB Cartridge

The New WRB Cartridge Configuration Form allows you to define configuration parameters for a new WRB cartridge. To use this form, you must have used the Web Request Broker Administration Form to select a Web Listener process and have clicked its Configure button. From there, you must have followed the Configure a New WRB Cartridge link.

Configuring a New WRB Cartridge

 To configure a new WRB cartridge, you must define at least one entry in both the Applications and Objects and Applications and Directories sections of the WRB Cartridge Administration Form. You can do this before or after using this form to define parameters.

Defining Parameters for a WRB Cartridge

 First enter the cartridge name in the Name of New Cartridge field. Use the same short alphanumeric name that you use in your WRB Cartridge Administration Form entries.

You can then define parameters in the form of name-value pairs. Enter the name of each parameter in the left column and its value in the right column. The interpretation of these parameters is entirely determined by your WRB cartridge; they have no meaning to any other WebServer component.

Note: If you run out of empty lines, click the Modify Cartridge Parameters button. This saves the information you have entered, and gives you five more blank lines.
