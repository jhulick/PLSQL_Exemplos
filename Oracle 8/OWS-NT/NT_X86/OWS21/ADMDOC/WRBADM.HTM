Configuring the Web Request Broker

The Web Request Broker Administration Form allows you to configure the behavior of the Web Request Broker (WRB) for a particular Web Listener process.

The WRB provides a high-throughput mechanism for Web Listener processes to use in fulfilling requests for dynamically generated hypermedia documents. See The Web Request Broker in "Chapter 1, WebServer Concepts" of the Oracle WebServer User's Guide for more information.

To configure WRB behavior for a particular Web Listener process, select the Web Listener process from the list on the Web Request Broker Administration Form and click its Configure button. This takes you to the WRB Cartridge Administration Form.
