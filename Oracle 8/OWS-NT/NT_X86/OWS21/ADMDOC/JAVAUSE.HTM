Using the Java™ Cartridge

The Java Cartridge is a part of the WRB that effectively interprets and executes Java on the server and sends HTML output to the client's browser as a Web page. To use the Java cartridge, you must have a Java Interpreter, which you can get from Sun Microsystems.

This section is divided into the following pages:

  • The Java Web Toolkit Overview. This section explains the Java classes provided with the Java cartridge.
  • Creating Java Wrappers for PL/SQL Packages (the pl2java Utility). The Java Cartridge interfaces to an Oracle database by executing PL/SQL packages contained in that database. This page explains the pl2java utility that automatically generates wrappers to interface to these packages.
  • Database Access from Java. This page explains how a Java application goes about interfacing to the database once the Java wrappers are created.
  • Dynamic HTML from Java. This page explains how to use Java Web Toolkit classes that make it easier for you to generate dynamic HTML from the Java cartridge.
