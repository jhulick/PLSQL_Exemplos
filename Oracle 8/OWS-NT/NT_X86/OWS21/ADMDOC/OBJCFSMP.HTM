Sample obj.conf File

This is a sample Netscape FastTrack server obj.conf file that enables the server to access Oracle WebServer components. The items in bold font are discussed in Notes on the sample file below. Italicized items are placeholders for values that you must provide, or values filled in by the Oracle WebServer installation process.

The sample file 

Init fn=flex-init access="NS_HOME/httpd-korangi/logs/access" format.access="%Ses->client.ip% - %Req->vars.auth-user% [%SYSDATE%] \"%Req->reqpb.clf-request%\" %Req->srvhdrs.clf-status% %Req->srvhdrs.content-length%"
Init fn=load-types mime-types=mime.types
Init fn=load-modules shlib=ORACLE_HOME/ows21/lib/oracle_nsapi.so funcs="oracle-adp-init,oracle-adp-auth-trans,oracle-adp-service,oracle-adp-addlog,oracle-adp-error,oracle-adp-name-trans,oracle-adp-path-check,oracle-adp-object-type"
Init fn=oracle-adp-init adapter="ORACLE_HOME/ows21/lib/adp_nsapi.so" cfgfile="ORACLE_HOME/ows21/admin/adp.cfg" serverconf="NS_HOME/httpd-korangi/config/obj.conf" servererrfile="NS_HOME/httpd-korangi/logs/errors" ORACLE_HOME="ORACLE_HOME" OWS_APPFILE="ORACLE_HOME/ows21/admin/svserver-name.app" userdbdir="/private1/netscape/httpd/ns-home/userdb"

<Object name=default>
AuthTrans fn="oracle-adp-auth-trans"
NameTrans fn="oracle-adp-name-trans"
NameTrans fn=pfx2dir from=/ns-icons dir="NS_HOME/ns-icons"
NameTrans fn=pfx2dir from=/mc-icons dir="NS_HOME/ns-icons"
NameTrans fn=pfx2dir from=/ows-abin dir="ORACLE_HOME/ows21/admbin" name="cgi"
NameTrans fn=pfx2dir from=/ows-bin dir="ORACLE_HOME/ows21/bin" name="cgi"
NameTrans fn=pfx2dir from=/owa_dba dir="ORACLE_HOME/ows21/bin" name="cgi"
NameTrans fn=pfx2dir from=/ows-doc dir="ORACLE_HOME/ows21/doc"
NameTrans fn=pfx2dir from=/ows-adoc dir="ORACLE_HOME/ows21/admdoc"
NameTrans fn=pfx2dir from=/ows-img dir="ORACLE_HOME/ows21/img"
NameTrans fn=pfx2dir from=/tr-img dir="ORACLE_HOME/ows21/demo/img"
NameTrans fn=pfx2dir from=/sample/bin dir="ORACLE_HOME/ows21/sample/bin" name="cgi"
NameTrans fn=pfx2dir from=/sample dir="ORACLE_HOME/ows21/sample"
NameTrans fn=pfx2dir from=/netscape dir="NS_HOME/docs"
NameTrans fn=document-root root="ORACLE_HOME/ows21/doc"
PathCheck fn=unix-uri-clean
PathCheck fn=find-pathinfo
PathCheck fn=find-index index-names="index.html,home.html"
PathCheck fn="oracle-adp-path-check"
ObjectType fn="oracle-adp-object-type"
ObjectType fn=type-by-extension
#ObjectType fn=force-type type=text/plain
Service fn="oracle-adp-service"
Service method=(GET|HEAD) type=magnus-internal/imagemap fn=imagemap
Service method=(GET|HEAD) type=magnus-internal/directory fn=index-common
Service method=(GET|HEAD) type=*~magnus-internal/* fn=send-file
AddLog fn=flex-log name="access"
AddLog fn="oracle-adp-addlog"
</Object>

<Object name=cgi>
ObjectType fn=force-type type=magnus-internal/cgi
Service fn=send-cgi
</Object>

Notes on the sample file

  • ORACLE_HOME represents the current value of the ORACLE_HOME environment variable. In the sample file $ORACLE_HOME/ows21/admin/obj.conf, generated by the installation process, occurrences of ORACLE_HOME are replaced with the current value of ORACLE_HOME.
  • NS_HOME represents the current value of the NS_HOME environment variable. For example, if the current value of NS_HOME is /home/netscape, you should substitute /home/netscape for all occurrences of NS_HOME.
  • server-name represents the name of the Netscape FastTrack server to which the obj.conf file applies.
  • The AuthTrans directive given in the sample must be the only AuthTrans directive in the file.
  • The first NameTrans directive given in the sample must be the first NameTrans directive in the file.
  • The last PathCheck directive given in the sample must be the last PathCheck directive in the file.
  • The first two ObjectType directives given in the sample must be the only two ObjectType directives in the file.
  • The last ObjectType directive given in the sample must be commented out or deleted from the file.
  • The first Service directive given in the sample must be the first Service directive in the file.
