Modifying a WRB Cartridge Configuration

The WRB Cartridge Modification Form allows you to modify parameters you've defined for your WRB cartridge. To use this form, you must have used the Web Request Broker Administration Form to select a Web Listener process and have clicked its Configure button. From there, you must have selected an existing WRB Cartridge from the list and clicked its Configure button.

Modifying an Existing WRB Cartridge

 To modify an existing WRB cartridge configuration, you can change its entries in the Applications and Objects and Applications and Directories sections of the WRB Cartridge Administration Form. You can also use this form to modify configuration parameters specific to the selected WRB cartridge.

Modifying WRB Cartridge Parameters

 You can modify the cartridge-specific parameters on the form and click the Modify Cartridge Parameters button at any time to apply your changes. You can also click the Delete Cartridge Configuration button at any time to delete all cartridge-specific parameters. The interpretation of these parameters is entirely determined by your WRB cartridge; they have no meaning to any other WebServer component.

Note: If you run out of empty lines, click the Modify Cartridge Parameters button. This saves the information you have entered, and gives you five more blank lines.
