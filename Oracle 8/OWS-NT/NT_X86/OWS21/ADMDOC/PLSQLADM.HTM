Configuring the PL/SQL Agent

The PL/SQL Agent Administration Form allows you to create and modify Database Connection Descriptors (DCDs). DCDs specify how the PL/SQL Agent accesses databases to fulfill HTTP requests.

There should always be at least two DCDs configured for any WebServer installation. The administration DCD allows the PL/SQL Agent to connect to a database to perform administration tasks, such as creating new database users. The PL/SQL Agent uses the default DCD to handle PL/SQL Agent requests that do not specify a DCD. You create these DCDs immediately after you install your WebServer.

Creating a New DCD

 To create a new DCD, follow the Create New Service link on the PL/SQL Agent Administration Form, which takes you to the Oracle Database Connection Descriptor Creation Form.

Modifying an Existing DCD

 To modify an existing DCD, select it from the list on the PL/SQL Agent Administration Form, and click its Configure button. This takes you to the Oracle Database Connection Descriptor Modification Form.
