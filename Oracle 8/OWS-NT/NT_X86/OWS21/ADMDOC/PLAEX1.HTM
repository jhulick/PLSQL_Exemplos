General PL/SQL Web Page Example

The following program shows how to use functions from the PL/SQL Web Toolkit to generate a simple dynamic Web page. It illustrates how the functions from the toolkit work together.

CREATE OR REPLACE PROCEDURE sample1

IS

   v_date DATE;

BEGIN

The following section of code generates the basic tags to open the page, provide it with the title "Hello World!", import a graphic, and print a header.

   htp.htmlOpen;

   htp.headOpen;

   htp.title('Hello World!');

   htp.headClose;

   htp.bodyOpen;

   htp.line;

   htp.img('/ows-img/ows.gif');

   htp.header(1,'Hello World!');

   htp.line;




The following section of code shows how the htp.print procedure can combine database variables like Sysdate with text strings and HTML tags. The htp.print procedure is useful for generating HTML that is not easily produced with the other toolkit procedures.



   htp.paragraph;




   htp.print(To_char(Sysdate, 'DAY') || 'is my favorite day of the week.');








   htp.paragraph;




   htp.print('You can still even use tags that are not in the HTP/HTF library by just including them <BLINK>directly</BLINK> in the text.');








   htp.paragraph;




   htp.paragraph;



The following section of code provide a link to the PL/SQL source code for this page. The text of the link is "Show Source", and the target is a URL that invokes an application called /sample/bin/show-sample-source.sh and passes the argument /owa/simple.sql to the parameter FILENAME. The source code returned is, of course, different from the source code that you could view in your browser, which is the HTML produced by the PL/SQL.



   htp.anchor('/sample/bin/show-sample-source.sh?FILENAME=/owa/simple.sql','Show Source');




   htp.line;








   htp.address('(c)1996 Oracle Corporation');




   




   htp.bodyClose;




   htp.htmlClose;




END;

show errors
