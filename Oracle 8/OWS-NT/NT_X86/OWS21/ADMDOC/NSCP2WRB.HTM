CONFIGURE the Netscape FastTrack/Enterprise Server for Oracle Web Request Broker 2.1


This page will:

  • configure the Netscape FastTrack/Enterprise Server configuration file obj.conf under $NS_HOME/http[ds]-[server_name]/config for Oracle Web Request Broker 2.1 and save the original obj.conf into obj.conf.sav.
  • create a default sv[server_name].app under $ORACLE_HOME/ows21/admin


Before you proceed, you MUST:
  • register your Netscape FastTrack/Enterprise Server to be configured with Oracle External Listener Registration.
  • install the Oracle Web Server 2.1 on the same machine where you run the Netscape FastTrack/Enterprise Server.
  • have created a Netscape FastTrack/Enterprise Server to be configured.
  • stop the Netscape FastTrack/Enterprise Server before doing the following configuration.
  • be sure that the currently Netscape server name is not currently in use by Oracle Web Server.
