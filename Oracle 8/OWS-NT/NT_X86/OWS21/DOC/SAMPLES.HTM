Note: The sample applications are intended to be run from the administrative listener (typically port 9999). If you wish to run them from another listener you must add the appropriate directory mapping entries that are contained in the administrative listener configuration file.

PL/SQL Agent Sample

 

NOTE: Before running any of the following Web Agent samples, you first need to install the Web Agent sample database packages first. To install these packages :
  • Create the DCD's for accessing the database, if you haven't done this already.
  • Using server manager (svrmgr23),connect to the Oracle Account corresponding to the owa_dba DCD (by default the database user www_dba).
  • Run the scripts "owains.sql" and "inst.sql":
SVRMGR>@%OWS21%\admin\owains.sql
SVRMGR>@%OWS21%\sample\owa\inst.sql
 
NOTE: If you wish to rerun inst.sql, please run dinst.sql first.

HTTP Cookie
Oracle WebServer implements HTTP Cookies and this sample will show you how to use them from the PL/SQL agent. Cookies are a way of storing state on the client for future visits to a web site. For instance, a shopping site could keep the items "in your shopping basket" in an HTTP Cookie until you decide to place your order.
