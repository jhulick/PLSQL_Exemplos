Versão 1.0, última actualização 14/8/97

Este ficheiro documenta o conteúdo do CD que acompanha o manual Oracle8: Programação em PL/SQL, de Scott Urman.  Há três itens diferentes neste CD:

  1. Código de origem para os exemplos utilizados no livro.
  2. Uma versão de demonstração do SQL-Station First Edition para Windows 95 and
     Windows NT, da Platinum Technologies.  Esta versão é completamente 
     funcional, mas expira ao fim de 30 dias.
  3. Uma versão de demonstração do Oracle Web Server, versão 2.1, para Windows
     NT e sistemas Solaris.  Esta versão é completamente funcional, mas expira 
     ao fim de 90 dias.

O CD tem a seguinte organização:
readme.txt    Este ficheiro readme.
code/         O directório code contém o código de origem para todos os exemplos
              utilizados no livro. No subdirectório code há um directório 
              distinto para cada capítulo, que contém os exemplos referentes
              a esse capítulo. Além disso, no directório code encontra-se um
              ficheiro readme.txt que contém uma descrição completa do conteúdo.
              O ficheiro code/ch01/tables.sql criará as tabelas exemplificativas
              utilizadas para os exemplos. O código é distribuído sob a forma de
              ficheiros de texto ASCII e, por essa razão, pode ser lido com um 
              editor de texto standard nos sistemas Windows e Unix.
station/      O directório station contém uma versão de demonstração do 
              SQL-Station 1.1, da Platinum Technology, para Windows 95 e Windows
              NT. Para o instalar, execute o programa setup.exe que se encontra
              no directório station. Pode consultar a documentação interactiva 
              (em formato HTML) no directório station/info. Para obter mais
              informações sobre o SQL-Station, incluindo preçários e informações
              sobre a forma de encomendar, consulte o local na Web da
              Platinum em http://www.platinum.com. O SQL-Station é analisado nos 
              Capítulos 13, 14, 19 e 22 deste livro.
ows-nt/       Estes directórios contêm versões de demonstração do Oracle Web
              Server.

ows-sol/      2.1. O directório ows-nt contém a versão para NT e o directório
              ows-sol contém a versão para Solaris. Para instalar a versão para
              NT, execute setup.exe no directório. Após a instalação, deve
              instalar a versão de correcção para a versão 2.1.0.3.2 que se
              encontra no directório ows-nt/patch. Consulte o ficheiro
              readme.txt do directório ows-nt/patch para obter mais informações.
              Para instalar a versão para Solaris, siga as instruções do
              ficheiro install.txt, que se encontra no directório ows-sol. Para
              obter mais informações sobre o Oracle Web Server, incluindo 
              preçários e informações sobre a forma de encomendar, consulte o 
              local da Oracle na Web em http://www.oracle.com. O Oracle Web 
              Server encontra-se descrito no Capítulo 19 deste livro.
