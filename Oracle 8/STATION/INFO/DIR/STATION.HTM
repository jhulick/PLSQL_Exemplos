PLATINUM SQL-Station

CURRENT VERSION: First Edition 1.1.1
TECHNICAL REQUIREMENTS: Supports Windows 95 and Windows NT (32-bit). SQL-Station is compatible with Oracle V7 or above.
DATE OF 1ST RELEASE: December 1996 (component tools released individually at earlier dates)
DATE OF LATEST RELEASE: June 1997

PRODUCT CATEGORY: Application Lifecycle Solutions

PLATINUM SQL-Station provides a comprehensive, integrated development environment to create, edit, test, debug, tune, and manage server side database code. Currently, SQL-Station contains three component tools: SQL-Station Coder, SQL-Station Debugger, and SQL-Station Plan Analyzer. SQL-Station Coder is a tool for developing and manipulating database server-side code and objects. With point-and-click ease, users can view an object's reverse engineered code, view and update table data in an editable spreadsheet, execute queries and other objects with a fast engine, and view all objects that reference or are referenced by another object. SQL-Station Debugger is the only Windows-based tool that debugs PL/SQL stored procedures, triggers, packages, and functions. SQL-Station Debugger supports debugging of objects which can be accessed either directly from the database catalog or from an operating system file. The tool features standard debugging operations such as breakpoints, variable watch, step in/step over, and call trees in a powerful, highly visual user interface. SQL-Station Plan Analyzer is a Windows-based tool that automates SQL capture and tuning for Oracle databases. SQL-Station Plan Analyzer significantly reduces the effort required to create, display, interpret, and test SQL optimization plans. With SQL-Station, developers can move through seamlessly integrated components with ease, facilitating efficiency and productivity.

PRODUCT CUSTOMERS
  • Application Programmers
  • Database Administrators (DBAs)

TYPICAL APPLICATIONS

  • Company wants to take advantage of the benefits of server-side objects and needs tools to develop, view, edit, and execute them from a familiar visual Windows environment.
  • Company needs to develop a mission critical application quickly. With SQL-Station's intuitive GUI, the team will be able to create, test, and debug their development database quickly.
  • DBA notices that a production procedure is suddenly classified as invalid by a server. Using SQL-Station Debugger, the DBA determines that a called procedure was changed resulting in an invalid datatype specification. With SQL-Station, the DBA quickly fixes the datatype and deploys the correct procedure.
  • Application programmer identifies a slow transaction from a client and needs to know the SQL the transaction uses. After identifying the SQL, the programmer needs assistance in tuning the statement.

BENEFITS

  • Integrates developer tasks into a single environment, encouraging testing, debugging, and tuning of database code at every stage of development.
  • Enables effective debugging of PL/SQL server-side code, thus reducing the time required to deliver reliable code.
  • Allows developers to use advanced client-side code debugging techniques for server-side code—features unavailable in any other tool.
  • Increases programming productivity with an advanced database catalog browser for reverse engineering and lookup, a query facility, and syntax checking.
  • Provides a transparent interface with server-side database code, which facilitates the maintenance of database objects.
  • Provides a single look and feel to all components, reducing the learning curve and increasing early productivity.

ASSOCIATED PRODUCTS: As part of PLATINUM's Application Lifecycle Solutions, SQL-Station can be integrated with products such as PLATINUM Paradigm Plus, an object-oriented analysis and design tool, and PLATINUM CCC/Harvest, a change and configuration management tool, to deliver comprehensive application development solutions.
