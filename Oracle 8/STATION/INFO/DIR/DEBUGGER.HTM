PLATINUM SQL-Station Debugger

CURRENT VERSION: 2.1
TECHNICAL REQUIREMENTS: Supports Windows 95 and Windows NT (32-bit). Compatible with Oracle V7 and above. Works in conjunction with SQL-Station Coder
DATE OF 1ST RELEASE: August 1996 (originally released as PLATINUM Watch-PL)
DATE OF LATEST RELEASE: June 1997

PRODUCT CATEGORY: Application Lifecycle Solutions

PLATINUM SQL-Station Debugger is the only Windows-based tool that debugs PL/SQL stored procedures, triggers, packages, and functions. SQL-Station Debugger supports debugging of objects which can be accessed either directly from the database catalog or from an operating system file. The tool features standard debugging operations such as breakpoints, variable watch, step in/step over, and call trees in a powerful, highly visual user interface. Developers also benefit from the powerful catalog browser for navigating through the database and from the reverse-engineering capabilities of SQL-Station Coder.

PRODUCT CUSTOMERS
  • Application Programmers
  • Database Administrators (DBAs)

TYPICAL APPLICATIONS

  • Developer of a stored procedure or package needs to verify that it functions as expected under various conditions and with different input values
  • DBA discovers that a package is not inserting values into a table consistently, and wants to review the call stack to identify the incorrect code in a procedure referenced by the package.
  • Application programmer discovers that a function is not returning the correct result when executed, and wants to quickly identify its flaws.
  • DBA notices that a production procedure is suddenly classified as invalid by a server. Using SQL-Station Debugger, the DBA determines that a called procedure was changed resulting in an invalid datatype specification. With SQL-Station Debugger, the DBA quickly fixes the datatype and deploys the correct procedure.
  • Application programmer needs to determine why a trigger is producing invalid values for a database table and needs to watch variable values in the trigger.

BENEFITS

  • Enables effective debugging of PL/SQL server-side code, reducing the time required to deliver reliable code.
  • Allows programmers to quickly fix all the impacts of programming errors by enabling them to determine and update all objects to which a procedure refers.
  • Allows developers to use advanced client-side code debugging techniques for server-side code—features unavailable in any other tool.
  • Uses innovative and powerful server-side technology to support complex debugging processes like expression evaluation and display of a complete call tree.

ASSOCIATED PRODUCTS: Integrated into SQL-Station, this component provides immediate debugging capabilities for objects displayed in SQL-Station Coder's catalog browser. The SQL-Station product suite also includes SQL-Station Plan Analyzer to ensure higher quality and better performing code by allowing the programmer to highlight SQL statements and check their optimization plans.
