PLATINUM SQL-Station Plan Analyzer

CURRENT VERSION: 2.3
TECHNICAL REQUIREMENTS: Supports Windows 3.1 or above, Windows 95 and Windows NT. Compatible with Oracle V7 or above.
DATE OF 1ST RELEASE: March 1996
DATE OF LATEST RELEASE: June 1997

PRODUCT CATEGORY: Application Lifecycle Solutions

PLATINUM SQL-Station Plan Analyzer is a GUI-based tool that automates SQL capture and tuning for Oracle databases. Plan Analyzer significantly reduces the effort required to create, display, interpret, and test SQL optimization plans. Additionally, an automated capture facility allows a DBA or application programmer to identify SQL statements that consume extensive resources on the server and, therefore, will readily benefit from performance tuning.

PRODUCT CUSTOMERS
  • Application Programmers
  • Database Administrators (DBAs)

TYPICAL APPLICATIONS

  • Developer using SQL to access an Oracle database wants to use the most efficient statement to perform a certain operation. Plan Analyzer will automatically select the appropriate hints.
  • Application programmer requires performance statistics to analyze why an application process is taking longer than expected. After identifying the SQL statement and associated timing statistics, the programmer needs assistance in creating and determining the optimization hints and other code changes to improve performance.
  • DBA needs to identify what SQL statement is using the most CPU, logical reads, or physical reads.
  • Application programmer identifies a slow transaction from a client and needs to know the SQL the transaction uses. After identifying the SQL, the programmer needs assistance in tuning the statement.
  • DBA needs to identify unused indexes and frequently used queries that sort or join data for attributes that do not have an associated index.

BENEFITS

  • Improves application and database server performance by enabling application programmers to tune their SQL statements.
  • Allows both novice and expert DBAs and programmers to perform sophisticated SQL optimization functions with an intuitive GUI and explanations of optimization plans in plain language.
  • Supports the rapid implementation of highly beneficial, but less frequently used, optimization techniques like hints and array binding.
  • Provides for easier tracking of SQL changes by maintaining a repository of SQL statements, and production optimization plans and alternatives.
  • Emulates the production execution environment for optimization of SQL statements.

ASSOCIATED PRODUCTS: SQL-Station Plan Analyzer provides tuning and analyzing capabilities to users of PLATINUM SQL-Station. When Plan Analyzer is combined with PLATINUM TSreorg, the tools provide extensive server performance improvement by tuning both the SQL used by the application and the physical implementation of the database (e.g., reorganizing tablespaces).
