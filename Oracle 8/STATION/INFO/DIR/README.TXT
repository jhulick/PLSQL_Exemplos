Instalação do SQL-Station:

Seleccione "setup.exe" com o rato no directório Station. Ser-lhe-ão fornecidas indicações para o processo de configuração, permitindo-lhe instalar cada componente do SQL-Station por um período de avaliação de 30 dias. Note que o código de servidor dos componentes tem de estar instalado para os produtos funcionarem correctamente. Para o SQL-Station Coder e Debugger só são necessários privilégios de criação para concluir a instalação. O SQL-Station Plan Analyzer necessita de privilégios de DBA para concluir a instalação.

A documentação interactiva encontra-se no directório /doc. Estes livros são apresentados em formato *.pdf, que pode ser lido com o Acrobat Reader. Juntamente com o software, foi incluída uma cópia do Acrobat Reader no directório /acrowin.

Integração do SQL-Station com o Visual Studio:

1. Abra o Visual Studio.
2. Seleccione Tools/Customize.
3. Seleccione o separador Tools na caixa de diálogo Customize.
4. Vá para o final da lista para acrescentar um novo item de menu.
5. Escreva &SQL-Station (a letra a seguir ao "&" tornar-se-á a tecla de aceleração).
6. Quando lhe for pedido o comando, seleccione "browse".
7. Vá para a localização até onde está instalado o SQL-Station e seleccione "Coder/Station.exe"
8. Não seleccione nenhuma das caixas de selecção da caixa de diálogo antes de terminar.
