PLATINUM SQL-Station Coder

CURRENT VERSION: 2.1
TECHNICAL REQUIREMENTS: Supports Windows 95 and Windows NT (32-bit). Compatible with Oracle, Microsoft SQL Server, Sybase SQL Server, and ODBC compliant data sources.
DATE OF 1ST RELEASE: October 1994 (originally released as PLATINUM SQL-Coder)
DATE OF LATEST RELEASE: June 1997

PRODUCT CATEGORY: Application Lifecycle Solutions
PLATINUM SQL-Station Coder is a tool for developing and manipulating database server-side code and objects. With point-and-click ease, users can view an object's reverse engineered code, view and update table data in an editable spreadsheet, and view all objects that reference or are referenced by another object. SQL-Station Coder enhances developer productivity with features such as editable code templates and code object generation wizards. The editor includes advanced features such as syntax highlighting and spell-checking, as well as standard editing facilities, such as graphical grep, search and replace, and bracket matching. Look-up tables allow users to drag-and-drop table columns and procedure calls into the code. Additionally, SQL-Station Coder possesses efficient execution and reporting capabilities. It is the core development environment for SQL-Station and smoothly integrates with SQL-Station's other components, SQL-Station Debugger and SQL-Station Plan Analyzer.

PRODUCT CUSTOMERS
  • Application Programmers
  • Database Administrators (DBAs)

TYPICAL APPLICATIONS

  • Team of database programmers working in a distributed environment is looking for a standardized query, execution, and programming tool.
  • Company wants to take advantage of the benefits of server-side objects and needs tools to develop, view, edit, and execute them from a familiar visual Windows environment.
  • Company is in the early stages of client/server RDBMS implementation and wants to set up application standards.

BENEFITS

  • Facilitates the development and maintenance of code objects by providing sophisticated editing, code generation, and execution functions.
  • Ensures reliability of server objects by encouraging testing prior to deployment.
  • Improves SQL programmer productivity by providing a fast and easy-to-use execution capability with reverse engineering of existing database catalog objects.
  • Simplifies all the procedures of accessing server-side code with an intuitive Windows interface.
  • Standardizes the query command tool interface for an environment where multiple database server types exist.

ASSOCIATED PRODUCTS: As a component of the SQL-Station product suite, SQL-Station Coder provides the development environment for producing database objects, which can be debugged with SQL-Station Debugger. SQL-Station also includes SQL-Station Plan Analyzer to ensure higher quality and better performing code.
