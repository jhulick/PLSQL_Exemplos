Instalação do SQL-Station:

Seleccione "setup.exe" com o rato no directório Station. Ser-lhe-ão fornecidas indicações para o processo de configuração, permitindo-lhe instalar cada componente do SQL-Station por um período de avaliação de 30 dias. Note que o código de servidor dos componentes tem de estar instalado para os produtos funcionarem correctamente. SQL-Station Coder e Debugger requerem acesso a uma ID de utilizador com o perfil de grupo DBA para a instalação ser concluída. O SQL-Station Plan Analyzer necessita de acesso à ID de utilizador SYS para concluir a instalação.

A documentação interactiva encontra-se no directório /doc. Estes livros são apresentados em formato *.pdf, que pode ser lido com o Acrobat Reader. Juntamente com o software, foi incluída uma cópia do Acrobat Reader no directório /acrowin.
